# PHILCOMP Logo Collection 2026
Copyright 2025 Philosophy Of Computing Research Group [philcomp.org](https://philcomp.org)

This revision features the following logo configurations:
- `full` is the default logo and the most used version.
- `long` is a variant of `full` designed for fullwidth applications, like in webpage landscape headers, footers with just the logo, etc.
- `trademark` simplies the the full name of the organization into just "PHILCOMP", it should be used in memorabilia and commercial applications only.
- `icon`, with just the iconic trademark alone. It must be used for reducted applications only.

A favicon is also included.

## Details
- `full` and `long` feature Spanish editions, marked with a `_es` suffix in the filename.
- Default logo is in the light scheme (#E3E2DB), dark version for contrast are also featured (#0F1017). **These mustn't be re-colored without the Brand Manager approval, please use the provided versions with official colors only**.
